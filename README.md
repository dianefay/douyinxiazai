# douyinxiazai
抖音视频网页下载 油猴脚本
原版为：douyin-user-data-download

0.5.6：初版发布，支持下载抖音用户主页数据、作品列表导出CSV、本地下载通道、GBK编码。

2.0.0：全新Pro版，新增浮动工具箱、批量选择与打包下载、下载历史记录、任务管理（暂停/继续/重置）。

2.0.5：修复图文作品的批量下载问题，恢复打包功能，优化图文处理逻辑。

2.1.1：自动过滤已下载作品，多线程并发下载，显示作品进度，超时重试，打包防崩溃，失败统计。

2.1.2：工具栏移至顶部居中，表格美化，修正文件名显示错误。

2.1.3：实时显示下载/打包进度百分比，详情栏动态展示当前文件名与打包进度条。

2.1.4：修复暂停/继续功能，重置按钮可强制清除所有任务状态。

2.1.5：新增高级设置面板，可调整打包阈值、并发数、ZIP分卷大小、重试次数与超时时间。

2.1.6：视频播放页新增内嵌下载按钮，支持直接下载当前播放的视频/图文。

2.1.7：工具栏默认收起为小圆条，自动吸附到搜索框右侧，拖拽后取消跟随，双击头部恢复吸附。

2.1.8：全面拦截XHR与fetch请求，智能扫描补漏按钮，增加周期补扫、手动补扫，设置项扩展工具栏默认展开、补扫间隔、提示时长。
